- Stackoverflow_survey.ipynb : main file (report) of the project
- functions.py : set of functions helping for the machine learning part
- texttable.py : set of function helping to create a table, found on github https://github.com/foutaise/texttable
- Data folder : contains the complete Stackoverflow survey
- survey_results_schema.csv : contains the questions of the survey

The dataset can also be found on : https://www.kaggle.com/stackoverflow/so-survey-2017/data
